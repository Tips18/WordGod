# 文档总览

当前 GitHub 仓库根目录只保留 `wordgod.apk` 和 `project/`。本目录内文档提到的开发相对路径，除特别说明外均以 `project/` 为项目根目录。

- `business/PRD.md`：产品目标、用户问题、功能范围和验收口径。
- `business/ITERATION_LOG.md`：迭代记录与关键决策。
- `technical/ARCHITECTURE.md`：系统架构、模块边界和数据流。
- `technical/DATABASE.md`：数据库模型与约束。
- `technical/CODE_STYLE.md`：编码风格与注释规范。
- `technical/CODE_NOTES.md`：实现细节、命令和重要说明。
