# 我不是词神 PRD

## 产品定位

我不是词神是一款面向已经完成阶段性背词学习的考研英语备考用户的工具型产品。产品核心不是教授新词，而是在真题语境中检验用户是否真正认识已经背过的词，并把识别不稳定的单词沉淀为高价值复习列表。

## V1 目标

- 用户可以进入首页并阅读一段来自考研英语一/二文章题库的真题自然段。
- 用户可以单击标记自己不认识或不熟悉的单词。
- 用户在本篇阅读中选中的生词会形成当前段落内的已选列表，便于边读边确认本篇待结算词，并可通过列表中的 `×` 按钮移除。
- 页面实时展示被标记词的词性、释义、原句和原句翻译。
- 被标记词的词性和中文释义优先来自已入库 `LexiconEntry`；数据库无匹配时回退 `词库/ecdict.md` 中的 ECDICT 词条；词典缺失或查不到时保留内容管线已有释义。
- 原句翻译必须是真实中文译文；当运行时翻译服务不可用时，页面应明确提示翻译暂不可用，不展示伪翻译占位。
- 用户点击下一篇时可进入新的段落检测流程。
- 已登录用户的标记会沉淀到生词本中，同一段内不重复计数，不同段重复标记会累计次数。
- 生词本按标记次数倒序展示，并保留最近三条上下文。

## V1 界面体验

- 前端界面采用阅读优先的“纸本文献感”设计，首页应让完整真题段落成为第一视觉焦点。
- 设计配色限定为以 `#E6DCCA`、`#FED8B6`、`#FFCFAB`、`#FFC2B0` 为主的暖色体系，标记词、主按钮和重点计数使用最强状态色。
- 被标记词的词性、释义、原句和翻译应以旁批式信息面板展示，避免打断正文阅读。
- 当页面宽度达到 `1280px` 及以上时，阅读页应让正文区域保持页面中央视觉焦点，本篇已选列表作为左侧旁批轨，Live Note 作为右侧固定旁批轨；低于该宽度时，本篇已选列表回到正文下方、Live Note 之前。
- 左右旁批轨面板不使用投影，只通过轻量纸面底色和细边框表达层级，避免在暖色背景上形成突兀阴影。
- 登录认证应作为全局弹窗承载，阅读拦截、生词本未登录拦截和顶部入口复用同一套纸张面板、细线分隔和暖色状态标签，不额外引入营销页式视觉结构。
- 游客点击“下一篇”触发的阅读拦截登录弹窗应提供右上角关闭按钮；关闭后停留在当前阅读页，不触发登录、不结算当前段落。
- 生词本未登录拦截弹窗应允许用户主动关闭；关闭后停留在生词本未登录提示页，不应因同一次未登录错误立刻重新弹出。
- 顶部标题栏应作为普通页面头部展示，不随滚动吸顶，并只保留“阅读检测”和“生词本”两个页面入口；游客显示“登录 / 注册”弹窗入口，已登录用户显示账号邮箱和“退出登录”操作。

## 用户流程

1. 用户进入首页并直接看到一段真题段落。
2. 用户点击段落中的单词完成标记和取消标记。
3. 用户查看本篇已选列表和词汇详情旁批。
4. 用户点击“下一篇”进入下一段。
5. 游客第一次点击“下一篇”时被要求登录或注册。
6. 登录成功后，当前段落的标记被同步并结算到生词本。
7. 用户进入生词本按优先级复习词汇。

## 功能范围

### 必做

- 邮箱密码登录与默认勾选的 30 天免登录；用户取消勾选时使用 24 小时短会话。
- 随机真题自然段分发。
- 单词标记与高亮反馈。
- 生词本聚合与详情查看。
- 白名单抓取器、段落切分、翻译和入库命令。
- 内部题库导入命令需要从 `真题题库/wordcram-kaoyan/articles/` 与 `真题题库/kaoyan-english-ii/articles/` 中读取考研英语一/二 Markdown，为每篇 `Text 1-4` 下的每个英文自然段生成一条可入库 `Passage`，并通过批量富化补齐句子翻译、lemma、词性和中文释义；英语二的 `Section I Use of English` 完形段落不进入随机阅读题库。
- `真题题库/kaoyan-english-ii/articles/` 保留考研英语二标准化文章资料，当前覆盖 2010、2013-2026 年，每年按 `Section I Use of English` 与 `Text 1-4` 输出自然段 Markdown；运行时只接入 `Text 1-4`，2011、2012 缺少本轮选定公开来源正文，需保留在索引失败记录中。
- WordCram 文章题库需保留来源 URL；缺失 Text 需写入抽取警告记录，V1 只纳入允许复用的公开来源。
- `词库/ecdict.md` 可作为 MIT 开源 ECDICT 英汉词典资料保留，并可通过内部命令导入 `LexiconEntry`，用于人工核对释义来源和运行时选词释义补全，不改变 V1 只支持考研英语题库的产品范围。

### 不做

- 商业化和支付。
- 邮件验证码与找回密码。
- 正式内容后台。
- 多考试题库前台筛选。
- 将 ECDICT 外部词典作为新的考试题库或独立前台词典查询功能。
- 授权不明、禁止批量抓取或禁止迁移的真题内容入库。
- 学习计划、社交、排行。

## 验收标准

- 首页能稳定展示可阅读段落。
- 本地开发启动应保证 `corepack pnpm dev:backend` 能生成并运行 Nest 入口 `backend/dist/main.js`，避免后端不可用导致首页无法加载段落。
- 本地开发前端应从仓库根目录 `.env` 读取 `VITE_API_BASE_URL`，当后端运行在 `3001` 等非 3000 端口时不应因回退到 `localhost:3000` 显示 `Failed to fetch`。
- 本地开发时，首页在 Vite 自动切换到 `localhost`、`127.0.0.1` 或 `::1` 的备用端口后仍应正常请求阅读接口，不应因 CORS 预检失败显示 `Failed to fetch`。
- 首页正文应展示完整英文段落，而不是只展示少量可点击 token。
- 本地未配置数据库存储模式时，首页应使用 `真题题库/wordcram-kaoyan/articles/` 与 `真题题库/kaoyan-english-ii/articles/` 抽取出的自然段题库，不应退化为只有两篇兜底样例，也不应因 PostgreSQL 不可用而返回 500。
- 首页段落接口不可用时，需要展示明确错误提示，不允许长期停留在加载状态。
- 标记词时会展示词性、释义、原句和原句翻译。
- 标记词时，本篇已选列表应按 lemma 去重展示当前段落内已选词面；取消标记或点击列表 `×` 按钮后，对应词面应从列表移除并取消本篇该词标记。
- 标记词的词性和释义应优先使用 `LexiconEntry` 中按 lemma 或表面词匹配到的结果，并在数据库无匹配时回退 ECDICT Markdown；匹配失败时阅读流程不报错。
- Live Note 不应展示“自动翻译：英文原文”或“逐句中文翻译将在内容富化入库后补齐”等占位文案。
- 运行时翻译 API 不可用时，阅读页仍可阅读和标记，原句翻译位置显示“翻译暂不可用，请稍后重试。”。
- 同一段同一词只计一次。
- 不同段重复标记会累加生词本计数。
- 生词本按标记次数倒序展示，详情可展示最近三条上下文。
- 游客第一次点击“下一篇”时被要求登录。
- 游客点击“下一篇”后应能关闭“登录后继续”弹窗并回到当前阅读页，当前段落和本地已选词保持可见。
- 已注册用户可以在登录弹窗直接使用邮箱密码登录；邮箱输入的大小写或首尾空格差异不应导致账号无法识别。
- 注册和登录提交空邮箱、格式不合法邮箱或空密码时应给出明确错误，不创建空账号，也不返回 500。
- 登录弹窗应显示“30天内记住登录”选项，默认勾选；取消勾选登录时刷新会话当天失效。
- 访问令牌失效但刷新会话仍有效时，`/auth/me` 应恢复当前用户并补发短期访问 Cookie。
- 当前会话已登录时，顶部导航不再显示“登录 / 注册”入口，而显示账号邮箱和“退出登录”按钮。
- 用户在全局登录弹窗或阅读拦截弹窗完成登录/注册后，顶部导航应立即切换为账号邮箱和“退出登录”，不依赖手动刷新页面。
- 内部内容导入只处理 `真题题库/wordcram-kaoyan/articles/` 中已确认来源的考研英语一资料，以及 `真题题库/kaoyan-english-ii/articles/` 中已确认公开来源的考研英语二 `Text 1-4` 资料；导入结果需要保留来源 URL，并能幂等更新同一试卷同一 Text 的多个自然段。
- 英语二资料接入 V1 运行时后不新增前台考试类型筛选，不导入 `Section I Use of English` 完形段落，也不绕过登录、付费、下载次数或复制限制获取内容。
- `词库/ecdict.md` 存在时不得被真题抽取、内存种子题库或内容入库流程当作考研英语 Markdown 文章处理。
- 后端主链路交付前应通过全量 ESLint，且不保留会影响质量基线的 error 或 warning。
# 2026-05-20 邮箱验证码认证更新

- V1 认证范围更新为：邮箱密码登录、邮箱验证码注册、邮箱验证码登录、邮箱验证码重置密码。
- 全局登录弹窗和阅读拦截弹窗需要保留邮箱密码登录，同时提供验证码登录入口。
- 注册前必须先发送并校验邮箱验证码，注册成功后直接进入已登录状态。
- 忘记密码通过邮箱验证码确认账号归属，再设置新密码，成功后直接进入已登录状态。
- 忘记密码发送验证码时，未注册邮箱应提示“邮箱未注册，请先注册”，不展示发送成功或倒计时状态。
- 邮箱验证码为 6 位数字，10 分钟有效，60 秒内不可重复发送，同一验证码最多尝试 5 次，验证成功后立即作废。
- 当前 V1 仍只支持考研英语题库，不因认证能力更新扩展考试类型、支付、OAuth 或后台 CMS。

# 2026-05-24 本地内存模式持久化更新

- 本地 `WORD_GOD_STORE=memory` 模式下，用户注册、登录刷新会话、邮箱验证码、生词本和阅读临时状态应在后端重启后继续保留，避免开发时因重启导致已注册账号无法继续接收登录验证码。
- 内存模式持久化仅服务本地开发和演示，不改变生产环境仍推荐使用 Prisma/PostgreSQL 的数据边界，也不扩展 V1 的题库、支付、OAuth 或后台 CMS 范围。
