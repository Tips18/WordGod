# AGENTS Rules

## 基础要求

- 每次写代码之前必须先通读 `docs/business/PRD.md`、`docs/technical/ARCHITECTURE.md`、`docs/technical/DATABASE.md`。
- 每次提交完成更改后必须同步更新 `README.md`、产品文档和技术细节文档。
- 每个函数必须带总结式注释，注释需要说明函数职责，不写空泛废话。

## 实施要求

- 阅读检测、生词本、登录认证是 V1 的最高优先级。
- 当前只支持考研英语题库，不要在 V1 中扩展到更多考试类型。
- 内容运营仅通过内部脚本完成，不要额外开发后台 CMS。
